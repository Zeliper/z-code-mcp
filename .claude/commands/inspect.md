# MCP Inspector

Launch the MCP Inspector to debug and test the MCP server.

## Steps

1. Ensure the project is built (`npm run build`)
2. Run `npm run inspector` to launch the inspector

## Notes

- Inspector opens at http://localhost:5173
- Use it to test MCP tools and view request/response
